# Cybersecurity-log-analyzer
A Python program that analyzes the login data to detect suspicious and high-risk users.

# Features
- Calculates total login attempts per user
- Detects users with unusual activity
- Identifies high-risk users based on login patterns
- Analyzes login locations

# Tools Used
- Python
- Pandas

# Output Example
- User A is suspicious
- User B is suspicious
  
- User A is HIGH RISK
